## ðŸŽ¨ Customization

To change the color scheme, edit the `:root` variables in the `<style>` block within `index.html`:

```css
:root {
    --bg-color: #0f172a;       /* Background */
    --accent: #6366f1;         /* Primary Action Color */
    --text-main: #f8fafc;      /* Primary Text */
}
```
